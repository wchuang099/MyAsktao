## 1.开发背景

### 1.1 开发约定

开发语言：python
ui框架：pyside2
多进程设计：采用Qt原生多进程from PySide2.QtCore import QProcess
图色插件: unify,(大漠同类软件，但插件原生支持vnc连接，无需单独通过vnc客户端)

### 1.2 核心架构

unify插件使用说明如下

- 游戏环境：游戏运行在虚拟机，每个虚拟机存在多个游戏窗口，通过开启vmware自带的vnc功能，连接vnc,进行字库图色识别unify.UNIFY(102) 102代表vnc端口 ，更多虚拟机 如103 104等等 
- 图色插件： unify是一款c++开发的虚拟机图色隔离插件，vnc方式连接虚拟机
- 键鼠操作：采用yjs易键鼠（双头盒子），其实unify也支持键鼠，但是存在漂移情况，只能采用硬件，每个虚拟机对应一个硬件 【端口 → 虚拟机 → 双头盒子 1对1全链路绑定】

### 1.3 unify插件介绍

插件特点：
unify是一款c++开发的虚拟机图色隔离插件，游戏运作在虚拟机，通过开启vmware自带的vnc功能，连接vnc,进行字库图色以及键鼠操作
该插件需要自行安装到主机c盘中，然后通过官方的库PyUnifyEN调用

#下面是利用unify写图色脚本的案例

```python
#unify.py
#导入python专用库
import PyUnifyEN
#创建实例
u = PyUnifyEN.Unify()
#加载dll,注意该文件在unify安装包里面的，已经安装到了系统盘里面，调用时也不需要指定路径
u.bsLoadDLL("Unify.dll")
#登录，注意unify密码只有一套，
u.bsLogin(acc,pwd)  
#版本
u.bsGetVer()
#连接虚拟机  虚拟机只用端口号连接即可，vnc无账号密码
u.bsConnect(vnc_port)
后续的图色键鼠都基于u实例
u.MoveTo()
u.FindPic()
```

### 1.4 多进程控制多台虚拟机

注意：unify非线程安全，一个进程只能安全地创建一个u = PyUnifyEN.Unify()实例
1.单虚拟机:
正常创建实例，连接一个 VNC即可
2.多虚拟机:
多进程方案：连接多个VNC，必须启动多个独立进程 —— 一进程、一实例、一 VNC（推荐方案）。
多线程方案： 单实例对多个vnc，（不采用此方案，如同多个人抢一个方向盘）

## 3. 基础框架搭建

### 1.1 目录结构设计理念

```
在开发正式脚本前，必须建立标准的资源管理规范。建议创建以下核心文件夹：
/资源 (Resources)：存放外部依赖，如大漠插件 (dm.dll)、第三方 DLL、图片素材、字库文件等。
/配置 (Config)：存放用户自定义设置、功能配置 (INI文件) 及用户缓存数据。
/代码 (src)
```

**关键设计点**：

1. **配置隔离**：每个账号使用INI的section隔离，如`[窗口_1]`、`[窗口_2]`
2. **模块独立**：每个功能模块单独文件，职责单一
3. **资源分类**：图片按用途分文件夹，字库按索引编号（0、1、2）
4. **数据类型化**：线程信息、任务数据等用类封装，避免全局变量污染

### 5.2 打图任务状态机详细流程

```python
# 状态机实现（Python伪代码）
def task_drawing(thread_id):
    flow_state = "读取当前任务"
    task_data = TaskData()
    
    while True:
        if flow_state == "读取当前任务":
            # 优先尝试快捷栏读取
            if read_task_from_quickbar(thread_id, task_data):
                logger.info(f"任务: {task_data.monster_name}, {task_data.map_name}")
                flow_state = "飞行"
            else:
                # 快捷栏无任务，打开任务面板
                flow_state = "查找赏金任务"
        
        elif flow_state == "查找赏金任务":
            if check_task_panel_open(thread_id):
                # 锁定识别范围（关键优化！）
                category_rect = get_task_category_rect(thread_id)
                
                # 查找"赏金任务"图标
                if dm_find_pic(thread_id, category_rect, "task_reward*.bmp", 0.9):
                    # 检查是否在打图任务详情页
                    detail_rect = get_task_detail_rect(thread_id)
                    if dm_find_pic(thread_id, detail_rect, "task_dig*.bmp", 0.9):
                        flow_state = "读取任务详情"
                    else:
                        # 点击赏金任务
                        click_position(thread_id, reward_pos.x + 20, reward_pos.y + 5)
                        dm_double_click(thread_id)
                else:
                    # 当前未接任务
                    close_task_panel(thread_id)
                    return False
            else:
                open_task_panel(thread_id)
        
        elif flow_state == "读取任务详情":
            # OCR识别任务详情
            task_content = dm_ocr_read(thread_id, task_detail_rect)
            
            # 解析多种任务文本格式（容错设计）
            if "埋他的" in task_content:
                # 格式: "挖宝XX正在地图名的X.Y埋他的宝贝"
                task_data.monster_name = extract_text(task_content, "挖宝", "正在")
                task_data.map_name = extract_text(task_content, "正在", "的")
                task_data.monster_x = extract_coord(task_content, task_data.map_name + "的", ".")
                task_data.monster_y = extract_coord(task_content, ".", "埋他")
                task_data.is_boss = False
            
            elif "前往制服" in task_content:
                # 贼王格式
                task_data.monster_name = extract_text(task_content, "一", "最近")
                task_data.map_name = extract_text(task_content, "最近在", "出现")
                task_data.is_boss = False
            
            # 验证数据完整性
            if task_data.monster_name and task_data.map_name:
                close_task_panel(thread_id)
                flow_state = "飞行"
            else:
                # 数据不完整，重新读取
                close_task_panel(thread_id)
                flow_state = "查找赏金任务"
        
        elif flow_state == "飞行":
            # 使用飞行旗快速传送
            if use_fly_flag(thread_id, task_data.map_name):
                flow_state = "寻路到指定地图"
            else:
                # 飞行旗失败，使用常规寻路
                flow_state = "寻路到指定地图"
        
        elif flow_state == "寻路到指定地图":
            if task_data.map_name == "建邺城":
                map_goto_datang_guojing(thread_id)
                flow_state = "移动到指定点"
            # ... 其他地图处理
        
        elif flow_state == "移动到指定点":
            map_move_to_point(thread_id, task_data.monster_x, 
                            task_data.monster_y, accuracy=5)
            flow_state = "点击强盗"
        
        elif flow_state == "点击强盗":
            click_npc(thread_id, task_data.monster_x, task_data.monster_y)
            flow_state = "对话并战斗"
        
        elif flow_state == "对话并战斗":
            if dialog_and_combat(thread_id, task_data):
                logger.info("任务完成")
                flow_state = "读取当前任务"  # 循环执行下一个任务
            else:
                logger.error("任务执行失败")
        
        time.sleep(0.5)
```
