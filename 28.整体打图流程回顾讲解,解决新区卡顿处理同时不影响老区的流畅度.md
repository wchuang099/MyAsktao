梦幻西游脚本开发笔记：打图任务全流程重构与优化

## 一、 任务架构概览 (Architecture Overview)

打图脚本不仅是简单的“点击”，而是一个由**任务链管理**、**智能寻路**、**图像识别 (OCR)** 及 **战斗逻辑** 组成的综合系统。本次重构重点在于解决“接任务、跨地图寻路、战斗判定”三个核心环节。

### 1. 打图主逻辑流

1. **状态自检**：检查是否已接任务（读取内存或 OCR 识别任务追踪栏）。
2. **接取任务**：若未接任务，执行“酒店店小二”接取逻辑。
3. **智能飞行**：根据目标地图名，优先调用“飞行旗”或“飞行符”进行就近传送。
4. **跨图寻路**：从传送点寻路至二级地图（如驿站 -> 国境 -> 强盗坐标）。
5. **交互与战斗**：定位强盗 NPC，执行点击对话与战斗。

## 二、 核心模块实现 (Core Modules)

### 1. 智能寻路与跨图逻辑 (Pathfinding)

为了提高效率，脚本需具备“就近飞行 + 跨图补偿”的能力。

- **飞行旗/符优先级**：
  - **国境任务**：长安酒店旗 -> 驿站老板 -> 大唐国境。
  - **江南野外**：长安城南门旗 -> 切换地图。
  - **建邺/傲来**：飞行符直达，随后执行地图内寻路。

### 2. 接任务逻辑：店小二交互

接任务不仅仅是点击 NPC，还需要处理 **视角拉近** 和 **抗干扰屏蔽**。

代码段

```
.子程序 接取打图任务
    // 1. 环境准备：屏蔽所有玩家（F9），防止NPC被遮挡
    模拟按键 (#F9键)
    延时 (500)
    
    // 2. 寻路至酒店内部坐标
    寻路至指定点 (26, 29) // 酒店店小二附近
    
    // 3. 图像识别 NPC
    .如果 (查找店小二 (x, y))
        // 视角补偿：若离NPC较远，先点击NPC侧方进行“拉近”
        点击坐标 (x - 50, y) 
        延时 (1000)
        // 再次点击 NPC 进行对话
        点击坐标 (x, y)
    .否则
        // 暴力点击：若未识别到，点击地图中心区域盲选
        点击坐标 (29, 17)
    .如果结束
```

### 3. 强盗识别与点击修正

由于强盗 NPC 会随机走动且存在遮挡，点击时需加入 **坐标修正 (Offset)**。

- **修正参数**：点击位置需在识别到的“强盗”名字标签基础上，Y 轴向下偏移（如人物腰部位置），确保触发对话框而非仅仅路过。
- **战斗判定**：通过识别战斗界面的“法术/道具”按钮位置，判断是否成功切入战斗。

## 三、 战斗与容错系统 (Battle & Fault Tolerance)

### 1. 战斗循环判定

采用独立循环监控战斗状态。

Python

```
# 战斗逻辑伪代码
def battle_loop():
    while is_in_battle():
        if can_action(): # 判定操作指令是否出现
            if check_hp() < 30: # 血量预警
                use_item("金创药")
            else:
                perform_attack() # 执行默认攻击或施法
        time.sleep(1)
    # 战斗结束，返回主流程
```

### 2. 常见问题处理 (Troubleshooting)

| **现象**         | **原因分析**         | **解决方案**                                  |
| ---------------- | -------------------- | --------------------------------------------- |
| **原地打转**     | 坐标误差过大         | 增大寻路到达的判定半径（如从3像素改为10像素） |
| **对话框不弹出** | 点击位置太偏         | 优化 NPC 点击偏移量，增加“视角拉近”动作       |
| **寻路卡死**     | 地图遮挡或虚拟机掉帧 | 引入“寻路超时”机制，超时未位移则重置飞行      |

## 四、 进阶扩展：贼王处理 (Zei Wang System)

贼王任务与强盗逻辑类似，但需注意：

1. **特征点提取**：贼王名字颜色与普通强盗不同，需独立配置字库。
2. **多点扫描**：贼王通常出现在固定范围内的几个坐标点，需执行“轮询寻路”扫描。
